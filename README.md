# 🍕 Pizzeria — Fullstack Food-Ordering App

A fullstack food-ordering web app for a pizzeria: browse the menu, place an
order, choose a payment method, and track its status — plus an admin dashboard
to manage products and orders.

> One of my first commissioned projects, early in my web-development journey —
> the one that opened my eyes to what's possible on the web. The original live
> site is no longer online; the demo below shows the app in action.

## ✨ Features

- 🛒 Order pizzas from the menu
- 💳 Choose a payment method — PayPal or cash on delivery
- 🔐 Cookie-based authentication
- 🧑‍💼 **Admin dashboard:**
  - Full CRUD on products (add, edit, remove pizzas)
  - Update order status for order tracking

## 🛠️ Tech Stack

- **Framework:** Next.js (fullstack)
- **Language:** TypeScript
- **State management:** Redux Toolkit
- **Database:** MongoDB
- **Image uploads:** Cloudinary
- **Auth:** cookie-based
